Project Overview  
This project is a Power BI dashboard created to analyze Adventure Works (AW) customer performance for the year 2026. The dashboard focuses on customer demographics, revenue distribution, and purchasing behavior to support data-driven business insights.

Key Metrics  
Total Customers: 18K  
Total Revenue: $307M  
Average Customer Age: 46  

Analysis Covered  
Revenue distribution by age group  
Customer segmentation based on family status (with and without children)  
Gender-based revenue comparison  
Identification of loyal, periodic, and VIP customers  
Top buyer contribution and customer concentration analysis  

Tools and Technologies  
Power BI Desktop  
DAX (Data Analysis Expressions)  
Data modeling using fact and dimension tables  
Interactive charts, KPIs, and filters  

Purpose  
This dashboard was developed as part of a business intelligence learning project and to showcase practical Power BI skills, including data modeling, DAX calculations, and interactive reporting.
